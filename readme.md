## 想定実行環境
* Windows 10 WSL2
    * Ubuntu 20.04

## 必要なモジュール
* nodejs
* npm
* sqlite3 →インストール方法： https://coffee-nominagara.com/wsl-ubuntu20-04-sqlite3-install

### npm でグローバル（-g）でインストールしておくもの
* yarn
* typescript
* ts-node

## 実行方法
``` bash
npm run serve
```
